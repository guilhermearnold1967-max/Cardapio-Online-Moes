## 📄 Descrição de Cada Arquivo

### 1. **cardapio-online.html** 
**O Arquivo Principal - COMECE AQUI**

- ✅ Arquivo único e autossuficiente
- ✅ ~4000 linhas de código HTML + CSS + JavaScript
- ✅ Funciona 100% no navegador
- ✅ Sem dependências externas
- ✅ Pronto para usar

**O que contém:**
- Header com logo e carrinho
- Grid de produtos responsivo
- Modal para adicionar produto
- Drawer (gaveta) para carrinho
- Checkout em 2 etapas
- Integração WhatsApp automática
- Persistência localStorage
- API ViaCEP integrada

**Como usar:**
```
1. Clique duplo para abrir no navegador
2. Procure "const WHATSAPP_NUMBER" (linha ~665)
3. Altere para seu número
4. Procure "const PRODUCTS" (linha ~668)
5. Altere produtos
6. Salve e compartilhe!
```
